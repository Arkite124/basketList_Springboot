### 이미지 사용 관련 안내

본 프로젝트에 사용된 모든 상품 이미지는 구글/네이버 쇼핑에서 수집된 공개 상품 홍보 이미지로,
**비상업적 포트폴리오 및 학습용 목적**으로만 사용되었으며,
상업적 목적이나 유료 서비스에 사용되지 않습니다.

이미지에 대한 저작권은 각 상품 제공처 및 브랜드에 귀속됩니다.
문제될 시 메일 주시면 해당이미지 확인 후 수정하도록 하겠습니다.
